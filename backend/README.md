

## Variáveis de ambiente recomendadas

```env
PROCESSOR_BACKEND=api
GOOGLE_API_KEY=coloca_a_tua_chave
GEMMA_MODEL_ID=gemma-4-26b-a4b-it
GOOGLE_AUDIO_MODEL_ID=gemini-2.5-flash
ALLOWED_ORIGIN=http://localhost:5173
```

Notas:
- `GEMMA_MODEL_ID` é usado para imagem.
- `GOOGLE_AUDIO_MODEL_ID` é usado para áudio.
- Também podes continuar a usar `GEMINI_API_KEY`; `GOOGLE_API_KEY` é apenas um alias mais claro.
